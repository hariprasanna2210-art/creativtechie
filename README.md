# CreativTechie Agency Website

Premium modern agency website for CreativTechie, built with Next.js 14 App Router, TypeScript, Tailwind CSS v4, GSAP, Framer Motion and Three.js.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```

## Structure

- `app/` - App Router pages, metadata, sitemap and robots.
- `components/` - Reusable layout, UI, homepage, blog and portfolio components.
- `lib/data.ts` - Service, portfolio, testimonial, FAQ and blog content.
- `app/globals.css` - Tailwind v4 import, design tokens and global utilities.
